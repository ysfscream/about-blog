# Git 使用

  • git init 初始化
  • git git remote add origin URL 添加到URL源代码库
  • git add -A
  • git commit -m "项目描述"
  • git push 推送
  • git pull 下拉更新
  • git push origin --delete <branchName> 删除远程分支
  • git checkout -b newBranch 新建 newBranch 分支并切换到 新建分支
    • 步骤类似于 git branch newBranch && git checkout newBranch
