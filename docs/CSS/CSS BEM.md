> BEM其实是块（block）、元素（element）、修饰符（modifier）的缩写，利用不同的区块，功能以及样式来给元素命名。这三个部分使用`__`与`--`连接（这里用两个而不是一个是为了留下用于块儿的命名）。



绿色部分为 block

蓝色部分为 element

红色部分为 modifier
